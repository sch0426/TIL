## 논리 타입

>참과 거짓을 의미한다
```java
boolean stop = true;
boolean stop = false;
```

### boolean 타입 변수
> 주로 두 가지 상태값을 저장할때 사용한다  
상태값에 따라 조건문과 제어문의 실행 흐름을 변경하는 데 사용된다
```java
int x = 10;
boolean result = (x == 20);
boolean result = (x != 20);
boolean result = (x > 20);
boolean result = (0 < x && x < 20);
boolean result = (x < 0 || x > 200);
```
